Representing a way to index into database rows

  • Crates using existing backends should use existing implementations of this traits. Diesel provides RowIndex<usize> and RowIndex<&str> for all bulit-in backends

  • Crates implementing custom backends need to provide RowIndex<usize> and RowIndex<&str> impls for their Row type.
